# FV Fitwear

Projeto front-end de e-commerce fitness.

## Tecnologias
- HTML
- CSS
- JavaScript

## Estrutura
- index.html = home
- produto.html = página do produto
- style.css = estilos globais
- produto.css = estilos da página do produto
- script.js = lógica do carrinho, busca e filtros
